Einbinden des Systems in den Tomcat-Server OHNE Verwendung von Eclipse
(Nur zum Kompilieren, aber nicht zum Ausführen).

Ziel für Webanwendungen:
- <Tomcat-installation-directory>/webapps/ROOT

Ziel für Kompilierte Klassen (.class):
- <Tomcat-installation-directory>/webapps/ROOT/WEB-INF/classes
Pfad muss zu Package passen:
- <Tomcat-installation-directory>/webapps/ROOT/WEB-INF/classes/com/sopra/team1723/ctrl

In der Datei <Tomcat-installation-directory>/webapps/ROOT/WEB-INF/web.xml muss man für jedes Servlet einen Eintrag hinzufügen.
(Zwischen den Web-App-Tags)

<servlet>
  <servlet-name>HelloWorld</servlet-name>
  <servlet-class>HelloWorld</servlet-class>
</servlet>

<servlet-mapping>
  <servlet-name>HelloWorld</servlet-name>
  <url-pattern>/HelloWorld</url-pattern>
</servlet-mapping>
